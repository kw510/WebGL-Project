For WebGL.html
	Keybinds
	x - rotate object about x axis
	y - rotate object about y axis
 	z - rotate object about z axis
	r - reset rotations
	e - edge render mode for object
	v - vertices render mode for object
	f - face render mode for object
	space - switch object to control

For Galaxy.html
	Keybinds
	up arrow - create a new solar system
